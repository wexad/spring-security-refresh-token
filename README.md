# Spring Boot JWT Authentication with Access and Refresh Tokens

## Overview

This Spring Boot application demonstrates how to implement JWT-based authentication using access and refresh tokens. It provides endpoints for user registration, login, access token generation, and refresh token generation.

## Features

- User registration
- User login with username and password
- JWT-based access token generation
- Refresh token for renewing access token
- Secure API endpoints that require valid tokens for access

## Technologies

- Spring Boot
- Spring Security
- JWT (JSON Web Tokens)
- Spring Data JPA
- H2/PostgreSQL database
- Maven

## Requirements

- Java 17 or higher
- Maven 3
- H2/PostgreSQL (for persistence)
